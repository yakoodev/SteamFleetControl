# Code of Conduct

## Our Commitment
We are committed to making participation in SteamFleetControl respectful, safe, and inclusive for everyone.

## Expected Behavior
- Be respectful and constructive.
- Assume good intent and ask clarifying questions.
- Give actionable technical feedback.
- Focus on improving the project, not attacking people.

## Unacceptable Behavior
- Harassment, threats, hate speech, or discrimination.
- Personal attacks, trolling, or repeated hostile behavior.
- Publishing private/sensitive data (credentials, tokens, personal info).
- Any behavior that creates an unsafe environment.

## Enforcement
Project maintainers may remove, edit, or reject comments, commits, issues, and pull requests that violate this Code of Conduct.

## Reporting
If you experience or witness unacceptable behavior, report it privately via security channel described in [SECURITY.md](SECURITY.md).

## Scope
This Code of Conduct applies to project spaces (issues, PRs, discussions, reviews) and official community communication channels.
